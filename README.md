# Goepon
冬季制作『ゴエポン』管理

## 学校での研究内容について
Visual StudioやUnreal Engine 5を活用した実践的なゲーム制作に注力してきました。
主にVisual Studioでのゲーム開発を行うことが多く、DxLibの関数を用いたコード設計を通じて、プログラムがどのように動作しているかという基礎の部分を伸ばしました。その応用として、
各種システムの制御やゲームループの構築などを自身で考えて一からコードを作成し、さらに理解を深めています。
この経験を活かし、今回のポートフォリオである冬の長期制作では、メインプログラマーとしてゲームループの処理やUI、ゲーム演出などの実装を担当しました。仕様変更に柔軟に対応できる拡張性の高い設計を意識するとともに、
ゲームが少しでも面白くなるようプログラマー以外の職種とも積極的に議論を重ね、作品全体のクオリティ向上に貢献しました。

## 技術的に困難だった課題と取り組み
冬の長期制作では、作品の第一印象を決める「タイトル画面のロゴ演出」のクオリティを高めるため、イージングを用いた演出制御システムの構築に注力しました。

## 直面した技術的課題
まず実装にあたり、授業で扱った資料を参考にして基礎的な関数を構築したのですが、表現したかった「タイトルロゴがバウンドしながら出現し、最後にピタッと気持ちよく止まる動き」のような、
緩急が激しく複雑な動作を実装できないという技術的な課題に直面しました。既存の単純なイージング数式だけでは理想の滑らかさや世界観に合わせた細部へのこだわりを表現しきれなかったため、
講師の方への積極的な質問や、イージングに関する技術サイトなどを活用して自発的に分析を重ねました。

## 試行錯誤と解決策
複雑な数式をプログラムの計算式へと落とし込むため、コードの条件分岐やパラメータ調整を何度も繰り返す試行錯誤を行いました。諦めずに挑戦を続けた結果、独自にカスタマイズした計算式を完成させることができ、
特にロゴが静止する瞬間の心地よい減衰と理想のバウンドの動きをシステムとして実装することに成功しました。

## 実装の成果と学び
このシステムによって、ゲームの世界観に完璧にマッチしたタイトル演出や、作品の魅力であるコメディ感を引き立てる画面構成を実現し、全体のクオリティ向上に大きく貢献できました。
この経験から、未知の技術に対し自発的に学びを深めることで自身の成長につながると改めて実感しました。
